# NeteaseMusic_Action
> 网易云刷歌脚本，原项目[地址](),在此基础上精简了一部分代码

## 使用方式

**fork 本项目**

在 Settins 下的 Secrets 添加 USER 和 PWD，USER 只能是手机号，不支持第三方账号登陆，PWD 就是密码
